# モノトーンUIテーマ（Bootstrapへの上書きレイヤー）

## 1. 目的・背景

標準のBootstrap（青基調）から、**無彩色（モノトーン）で落ち着いた見た目**に整えたい。色数を絞ることで「シフト管理の道具」としての可読性・上品さを優先する。Bootstrap本体やテンプレートの構造は変えず、**薄い上書きCSSを1枚重ねるだけ**で全画面に効くようにする（差し戻しも容易）。

## 2. 仕様（デザイン方針）

- **配色は無彩色**: 墨（`#18181b`）×グレー×紙のような背景（`#f3f2ef`）。アクセントは黒に近い墨。
- **色で伝えていた状態を、色以外で伝える**: バッジ/アラート/テーブルの「赤・緑・青・黄」を、**濃淡・塗り/枠線・アイコン・絵文字**で区別する。
  - 目立たせたいもの（募集中・提出済み・未読）＝**濃い墨の塗り**。
  - 中立（締切など）＝**淡いグレー**。
  - 要対応（未提出など）＝**白地＋黒枠** または **網掛け**。
  - フラッシュメッセージは先頭に記号（✓ ／ ！ ／ ⚠ ／ ・）を付けて成否が分かるようにする。
- **書体**: 本文＝Noto Sans JP、見出し（h1〜h4）・ロゴ＝Noto Serif JP（Google Fonts CDN）。
- **余白・角丸・淡い影**でカード/ボタンに立体感。並ぶカードはホバーでわずかに浮く。
- **スマホ優先**で可読性（行間・字間）を確保。

## 3. データモデルの変更

**なし**（見た目だけ）。マイグレーション不要。

## 4. 実装メモ

| ファイル | 変更 |
| --- | --- |
| `static/css/theme.css` | **新規**。Bootstrap 5.3 の上に重ねる上書きレイヤー。CSS変数（`--bs-btn-bg` 等）と要素セレクタで色・角丸・影・書体を上書き |
| `templates/base.html` | `<head>` に Noto フォントと `theme.css` を読み込み。`<body class="bg-light …">`、フラッシュメッセージに `app-msg` クラス、フッター（`app-footer`）を追加 |

### 仕組みのポイント
- Bootstrapはコンポーネントの色を**CSS変数**（`--bs-btn-bg`、`--bs-btn-hover-bg` など）で持っているので、**変数を上書きするだけ**で `.btn-primary` 等の見た目を変えられる（!important を多用しない）。
- 状態色（`.bg-success` / `.bg-danger` / `.table-danger` など）は、テンプレートの**クラス名は変えず**にCSS側で無彩色へ読み替える。テンプレートを触らずテーマだけ差し替えられる。
- フォント・CSSはCDN/静的配信。**本番では `collectstatic` が必要**（後述）。

## 5. アップデート手順（本番反映）

テンプレートと静的CSSの変更。モデル変更・新規依存はないので `migrate` は不要。**`theme.css` は静的ファイルなので `collectstatic` が必要**。

```bash
# ① 開発側でコミット＆push
git add static/css/theme.css templates/base.html
git commit -m "feat(ui): モノトーンテーマを追加"
git push origin main

# ② 本番(Docker)マシンで
git pull
docker compose up -d --build
docker compose run --rm web python manage.py collectstatic --noinput
```

> 静的ファイルは名前付きボリュームにあるため、`up --build` だけでは古い `theme.css` が残ることがある。必ず `collectstatic` を実行する（運用マニュアル第4部）。反映確認は `curl -I http://<host>/static/css/theme.css`（200）／ブラウザはハードリロード。

## 6. 確認項目

- 全画面でナビ/ボタンが黒基調、背景が紙色、見出しが明朝になっている。
- バッジ「募集中/締切/提出済み/未提出」が色なしでも区別できる。
- フラッシュメッセージ（成功/エラー/警告）に記号が付く。
- 未提出行（提出状況一覧）が網掛けで分かる。
- `static/css/theme.css` が 200 で配信される。
